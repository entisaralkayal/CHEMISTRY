# Chemistry Course Platform

Interactive chemistry course platform for university students.

## Stack

- React + Vite
- Node.js + Express
- SQLite for local server mode
- Netlify Functions + Netlify Blobs for Netlify deployment

## Features

- 7 chemistry curriculum chapters
- chapter explanations and learning objectives mapped to CLOs
- concept maps and reaction summaries
- chapter-end quizzes
- personalized feedback based on student answers
- teacher dashboard and exports
- virtual chemistry teacher chatbot

## Local development

```bash
npm install
npm run build
npm start
```
